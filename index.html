Consígalo en el Centro de aplicaciones

Obtenga aplicaciones gratuitas y de pago en el Centro de aplicaciones, la tienda donde el usuario decide la cantidad a pagar a desarrolladores independientes. Cada aplicación ha sido revisada por elementary para asegurar una experiencia nativa, segura y respetuosa con la privacidad.


Controles de los padres

  • Tiempo ante pantalla
    • Establezca límites de tiempo según el usuario para los días de la semana, fines de semana o ambos.
  • Uso de internet
    • Gestione las páginas web permitidas. Las reglas afectan a todas las aplicaciones para el usuario, incluso si se utiliza un navegador web diferente.
  • Gestione las aplicaciones
    • Elija qué aplicaciones son seguras para que accedan los más pequeños. Opcionalmente también puede permitir el acceso con su contraseña

Fáciles de usar. Ideales para un uso eficiente del teclado.

elementary OS está diseñado para que sea fácil de entender y de usar para el nuevo usuario. Esto no quiere decir que sea limitado; los potentes y personalizables atajos de teclado garantizan mantenerse productivo, ya sea el primer o el enésimo día que usa elementary.

Respeto por la privacidad. En
todos los ámbitos

Sus datos le pertenecen exclusivamente a usted. No hacemos ofertas de publicidad ni recopilamos datos personales. Nos financiamos directamente gracias a nuestros usuarios, que pagan lo que desean por elementary OS y por las aplicaciones del Centro de aplicaciones. Así es como se debe hacer.

Nuestra normativa de privacidad=>


Sin escuchas

elementary OS le a ayuda a controlar el funcionamiento de las aplicaciones. Cuando una aplicación use el micrófono, se muestra un indicador para informar acerca de ello. Cuando una aplicación utilice demasiada energía, el indicador de energía lo marcará.


Servicios de Ubicacion

Cuando una aplicación quiera acceder a su ubicación, tiene que pedirlo. Se muestra un mensaje que indica qué aplicación y la precisión que solicita. Siempre se puede revocar el acceso en Configuración del sistema ▸ Seguridad y privacidad.l


Mantenimiento

elementary OS puede mantener en orden los archivos temporales y los eliminados. Esto no solo conserva libre el almacenamiento de su ordenador, sino que ayuda a garantizar que los datos privados no vuelvan a ser molestos.
